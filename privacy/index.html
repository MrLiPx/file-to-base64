Privacy Policy

Introduction

Base64 Image Converter is an open-source tool hosted at mrlipx.github.io/base64-image-converter. We are committed to radical transparency about how the tool handles your data.

Data Collection & Processing

Local processing only

  • All image conversions are processed entirely in your browser using JavaScript
  • Images are decoded to Blob objects and previewed via URL.createObjectURL() — no data is ever sent anywhere
  • No images or Base64 strings are ever uploaded to any server
  • You can verify this by opening the browser Network tab — you will see zero requests when converting

No storage

  • We do not store any images, Base64 data, or conversion history
  • There is no server-side processing of any kind
  • All data exists only temporarily in your browser's memory and is discarded when you navigate away
  • Blob object URLs are revoked (URL.revokeObjectURL()) when no longer needed

No personal information

  • We do not collect any personal information whatsoever
  • No account creation, login, or registration is required or available
  • No email addresses, names, IP addresses, or identifiers are collected by this tool

Open Source Commitment

This tool is fully open source. Every line of code that runs in your browser is publicly visible. Privacy isn't a promise — it's verifiable code.
  • Licensed under the MIT License
  • Available on GitHub: github.com/MrLiPx/base64-image-converter
  • You can fork and self-host your own instance at any time

Analytics & Cookies

Analytics

  • We use only GitHub Pages built-in traffic stats — aggregated, anonymous page-view counts with no personal identifiers
  • No third-party analytics scripts are loaded (no Google Analytics, Mixpanel, etc.)

Cookies

  • We set no cookies of our own
  • GitHub Pages may set essential infrastructure cookies
  • No tracking, advertising, or third-party cookies

Security

  • All conversions happen locally — there is no server-side attack surface for your image data
  • The site is served over HTTPS via GitHub Pages with industry-standard TLS
  • Open-source code means any security issues can be identified and fixed publicly

Compliance

Because we collect no personal data and perform no server-side processing, the compliance footprint is minimal by design. This policy is intended to be compatible with GDPR, CCPA, PIPEDA, and other applicable privacy regulations.
